## 職務要約

大学卒業から現在に至るまで、 IT 業界でエンジニアとして従事。  
開発者としてコードを書く日々の傍らで、その時々で職務上必要となる技術や興味を持った分野に関する資格を取得し、体系的な知識の習得と参画するプロジェクトの業務効率の向上に努めている。  
ここ数年はシニア的な立ち位置のエンジニアとして、自身もコードを書きながら後輩エンジニアのサポートを行い、マネージャと共にシステム全体の設計や顧客との折衝も行っている。

直近一年半程 (2026/01 現在) はコードを書くことは少なくなったが、配下メンバ数名を管理するプレイングマネージャ (チームリーダー) としてプロジェクトへ参画。  
技術的には Microsoft Azure 上での開発・保守を行っており、クラウド基盤・ネットワーク・アプリケーションの設計・製造まで幅広くフルスタックに行っている。  
※ 現プロジェクトはスクラムを採用しているので、スクラムマスターではないが開発チームのサーバントリーダー的な立ち位置で動いている。

今後のキャリアとしては、クラウド領域での専門性をさらに高め、設計から運用まで一貫して携われるエンジニアを目指している。  
顧客とコミュニケーションを取りながら課題解決を進め、将来的には技術力とマネジメント力の双方を活かしてプロジェクトをリードできる存在として成長したい。
