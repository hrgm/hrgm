# About

## **_hrgm_**

フロントエンド寄りのフルスタックな Web エンジニアでしたが、現在はクラウドエンジニアとして活動中。

最近はコードを書く機会が減り、クラウドを利用したアプリ基盤構築や運用設計・保守などを行っていることもあって、クラウドアーキテクトやクラウド・プラットフォーム戦略コンサルを目指したいと思うようになりました。  
まずはこれまでのアプリ開発経験を活かして、アプリ基盤寄りのクラウドアーキテクトを目標にスキルとキャリアを磨いていこうと考えています。

過去の実績としては、アプリサーバ・データベースなどの比較的レイヤーが高めのミドル領域から、凝った Web デザインを除くフロントエンドまでを守備範囲とし、様々なプロジェクトへ参画してきました。  
現職の勤続が長いので、ここ数年はチームリーダーとしてメンバ管理・サポートなども行っています。

フロントエンドは Angular を使用した SPA や、 ES2015+ / TypeScript と Sass を webpack でトランスパイルして MPA の構築などをしていました。  
Web デザイン (UI / UX) は専門外ですが、何かしらの CSS フレームワークや UI ライブラリを使えば、それらしい業務アプリを作成可能です。

バックエンドは今まで PHP や VBScript (ASP) 、 Ruby on Rails など色々触ってきましたが、最近は Python に落ち着きました。  
もしくは、 Node.js で JavaScript (TypeScript) を書くかのどちらかです。

インフラやサーバ周りは一から環境構築・運用するほどの知識や技術を持ち合わせていないので、誰か詳しい人に任せるか、 Microsoft Azure や Firebase などのマネージドなクラウドサービスを利用するようにしています。  
前述のように最近は Microsoft Azure 上での開発・運用・保守を行うプロジェクトに参画しているので、もっぱら技術的な学びは Microsoft Azure に偏っています。

キャリアアップのためには、プライベートでも常に学び続ける必要があると考えており、それ自体にはまったく抵抗がありません。  
知識と技術のアップデートをし続けることが仕事の成功に繋がり、それが結果としてプライベートを充実させる一番の近道だと思っています。

お金と時間と良質な仕事を与えると、勝手にスキルアップします。

## Like - 好きなこと

一人で黙々と作業をするのが好きですが、気晴らしも兼ねてチームでわちゃわちゃするのも楽しいです。  
優秀なマネージャーがいるチームで、自分の役割として仕事とじっくり向き合うことができる環境だと最高です。

最近は小さなチームで自分の経験や知識を活かしてメンバのサポートを行い、チームとしてパフォーマンスを発揮することの楽しさも感じ始めました。

チームへ貢献するために自分のパフォーマンスを出す・高めることを第一に考えており、そのための自己研鑽には時間を惜しみません。  
表面的に技術を理解してさくっと使用するのではなく、書籍や公式ドキュメントをじっくり読みこみ、理解して使いこなすことに悦びを感じます。

**何でも満遍なくできる人** よりも **何かに尖った人** の方がカッコイイと思っています。

業務では常に自分以上のエンジニアを追いかけていたいです。  
最近はチームマネジメントやスクラム開発を行っていて、管理・支援系の業務にも興味を持ち始めています。

## Dislike - 好きじゃないこと

明らかに無駄だと分かっているのに、過去からの慣習や惰性で続けているだけというのはあまり好きではありません。

また、そうしたことへの改善や意見を気軽に言えないような組織・雰囲気は避けたいと思っています。  
チームメンバとは同じ方向を向いて協力して仕事を進めていきたいので、メンバを敵視したり、高圧的・非協力的な態度をとるメンバがいるチームだと、なかなかパフォーマンスは上げづらいです。

2020 年春頃からのコロナ渦によるリモートワークを通して、 QOL と業務効率の爆上げ (特に通勤時間を自己研鑽に充てられるのが大きい) を経験したので、もう意味もなく決まった時間に決まった場所で仕事をするのはやめようと思いました。

# Recent activities - 最近どう ?

2024 年後半から Microsoft Azure を使ったプロジェクトに参画し、開発手法も結構しっかりとしたアジャイル (スクラム) で行っています。  
プロジェクト遂行に必要な Microsoft Azure の知識・技術をしっかり学ぶために、各種資格の取得を進めています。

これまではエンジニアとして Web システムの開発ばかりしていましたが、ここ一年以上は自分もエンジニアとして動きつつ、リーダーとして一緒にプロジェクトに参画しているチームメンバの支援をするようになり、マネージメント系の仕事の楽しさやアジャイルの開発手法に興味を持ち始めました。

プライベートでは関西圏で行われる様々な LT 会や勉強会にちょくちょく参加していましたが、コロナ禍以降はほぼ参加することが無くなりました。  
最近はリモート中心ではありますが、また少しずつ参加する機会を増やしていこうと思っています。

また、日々学んだことは [はてなブログ](https://ai-can-fly.hateblo.jp/) などで、少しずつアウトプットするように心掛けています。

# Future Plans - 今後の予定

何となくこれまでのキャリアの延長線上に行き詰まりを感じていて、エンジニアとして開発を行うだけでなくマネージメントやコンサルティングにも業務の幅を広げていくことで活路を見出していけたらと考えるようになりました。

一旦のゴールとして、クラウド移行・導入やクラウド基盤の最適化などを行うクラウド・プラットフォーム戦略コンサルタントを目標に、まずはクラウドの知識を身に付け、これまでのアプリ開発の経験を活かしてアプリ基盤寄りのクラウドアーキテクトを目指していこうと思っています。

ただ、現職では様々な制約から上記キャリア形成が難しいため、中長期的な視点でキャリアアップと所属組織への貢献ができる環境を探しています。
